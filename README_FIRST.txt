How to use this ZIP

1. Extract this ZIP.
2. Copy fix-ai-tutor.cjs into your project root folder.
   Project root = folder where package.json is located.
3. Open terminal in that folder.
4. Run:

   node fix-ai-tutor.cjs
   npm run dev

5. Check:
   Demo Site → AI Tutor

After confirming it works:
- Upload the modified src/WorkingDemoApp.jsx to GitHub.
- Do not upload this ZIP file directly to GitHub, because GitHub will not automatically apply the fix from a ZIP.
