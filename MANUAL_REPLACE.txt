AI Tutor red box remove fix

- The paragraph inside the red-box area:
  "AI Tutor is currently turned off..."

Automatic method:
1. Extract this ZIP inside your project root folder.
2. Your project root means the folder where package.json is present.
3. Run:

   node fix-ai-tutor.cjs

4. Then test:

   npm run dev

5. Open Demo Site → AI Tutor.
