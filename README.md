# JEE Blueprint V12 - Topic Planner

This version adds topic-level lecture planner tracking.

Features:
- 11th IIT JEE, 12th IIT JEE, and Dropper planner topics
- Chapter click opens only topics from the uploaded planners
- Topic checkbox/tick system
- Accurate topic-based progress percentage
- Recent activities can be deleted one-by-one or cleared all
- English UI and mobile responsive layout
- Gemini AI tutor backend remains included

The uploaded planner PDFs were used only to extract chapter/topic/lecture labels. Faculty names, timings, and paid links are not included.
